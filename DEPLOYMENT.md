# Deployment Instructions

## Streamlit Cloud (Recommended)

1. Go to https://share.streamlit.io/
2. Click "New app"
3. Select repository: `YOUR_USERNAME/money-freedom`
4. Main file: `app.py`
5. Click "Deploy"

Your app will be live at:
`https://YOUR_USERNAME-money-freedom.streamlit.app`

## Local Testing
```bash
pip install -r requirements.txt
streamlit run app.py
```
